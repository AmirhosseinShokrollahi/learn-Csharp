.هر متغیر یک نام و نوع دارد که مشخص میکند چه داده ای میتوان در آن ذخیره کرد

.با استفاده از متغیر ها میتوانیم مقادیر مختلف را در حافظه ذخیره و مدیریت کنیم

:اصلی ترین و کاربردی ترین متغیر ها به ترتیب زیر هستند

int: برای ذخیره سازی اعداد صحیح استفاده میشود

double: برای ذخیره سازی اعداد اعشاری استفاده میشود

char: برای ذخیره سازی یک حرف یا کاراکتر استفاده میشود

string: برای ذخیره سازی کلمه یا متن استفاده میشود

bool: برای ذخیره سازی غلط یا درست بودن یک متغیر استفاده میشود
